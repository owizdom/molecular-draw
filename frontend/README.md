# Molecular Draw Frontend

React + TypeScript frontend with 3D molecular visualization.

## Setup

1. Install dependencies:
```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- 3D molecular visualization with React Three Fiber
- Interactive structure editing
- AI-powered chat interface
- Property display and calculation
- SMILES import/export
